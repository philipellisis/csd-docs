# Service Board

## Introduction

The Service Board (also called the Expansion Breakout Board) adds an additional 8 inputs to your PinOne setup, conveniently labeled for common service and Pinscape commands. It connects to the expansion board connector or directly to the main board and includes mounting brackets for cabinet installation.

This board makes it easy to add dedicated service buttons — such as coin, start, exit, and other cabinet controls — without using up inputs on your main board.

You can purchase the Service Board from the [Cleveland Software Design store](https://www.clevelandsoftwaredesign.com/pinball-parts/p/expansion-breakout-board-zx5y6).

## Specifications and Features

- 8 additional labeled button inputs
- Pre-labeled for common service and Pinscape commands
- Connects to the expansion board connector or main board via 20-pin ribbon cable
- Includes mounting brackets for cabinet installation
- Weight: 0.2 lbs
- Dimensions: 2" x 1" x 1"

## Compatibility

The Service Board is compatible with both the **PinOne Main Board** and the **PinOne Mini** board.

## Hooking it all up

Connecting the Service Board is straightforward — simply plug the 20-pin ribbon cable from the PinOne Main Board (or Mini) into the Service Board connector. The board is labeled with the function of each input to make wiring your buttons easy.
